# SimpThai

Nwjs Apps capable getting simple calculation result

# Usage

```
>npm install nw --nwjs_build_type=sdk
>npm install nw-builder --nwjs_build_type=sdk
```


# Run
```
npm run dev
```

# Build
```
npm run prod
```
